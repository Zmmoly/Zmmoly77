# التثبيت التلقائي لبيئة تطبيق زمولي للذكاء الاصطناعي

تم تحسين المشروع لتسهيل تثبيت المتطلبات وتنزيل نماذج الذكاء الاصطناعي تلقائياً، حتى للمطورين الذين لا يملكون Python أو المكتبات اللازمة.

## الطريقة السهلة - مهمة التثبيت

قمنا بإضافة سكريبتات تهيئة سهلة الاستخدام:

### لمستخدمي Windows
1. انقر نقراً مزدوجاً على الملف `app/script/setup_windows.bat`
2. سيقوم السكريبت بالتحقق من وجود Python وتقديم تعليمات للتثبيت إذا لم يكن موجوداً
3. سيقوم تلقائياً بتثبيت المكتبات المطلوبة

### لمستخدمي Linux/macOS
1. افتح Terminal وانتقل إلى مجلد المشروع
2. قم بتنفيذ:
   ```
   chmod +x app/script/setup_unix.sh
   ./app/script/setup_unix.sh
   ```
3. سيقوم السكريبت بالتحقق من وجود Python وتقديم تعليمات للتثبيت إذا لم يكن موجوداً
4. سيقوم تلقائياً بتثبيت المكتبات المطلوبة

## التنزيل التلقائي للنماذج

بعد تشغيل سكريبت التهيئة والتثبيت:

1. افتح المشروع في Android Studio
2. عند النقر على "Build" أو "Run" أو "Generate APK"، سيقوم النظام تلقائياً بـ:
   - التحقق من وجود المكتبات وتثبيتها إذا كانت مفقودة
   - تنزيل نماذج الذكاء الاصطناعي (بما في ذلك النموذج الكبير 128 ميجابايت)
   - وضع النماذج في المجلدات المناسبة
   - إظهار تقدم التنزيل في نافذة الـ Console

## ماذا يحدث في حالة عدم وجود Python؟

إذا لم يكن Python مثبتاً، سترى رسالة خطأ في Android Studio. لا تقلق، ببساطة:

1. قم بتثبيت Python 3.6 أو أحدث من [الموقع الرسمي](https://www.python.org/downloads/)
2. قم بتشغيل سكريبت التهيئة المناسب لنظام التشغيل الخاص بك
3. حاول البناء مرة أخرى

## ميزات النظام

- **التحقق من المتطلبات**: تلقائياً يتحقق من وجود Python والمكتبات
- **تثبيت المكتبات**: تلقائياً يثبت المكتبات المفقودة
- **التحقق من الإصدارات الموجودة**: تخطي تنزيل الملفات التي تم تنزيلها بالفعل
- **الاستكمال بعد الانقطاع**: في حالة انقطاع التنزيل، سيقوم بإعادة تنزيل الملفات غير المكتملة فقط
- **معلومات تفصيلية**: إظهار حالة كل عملية تنزيل ومعلومات تشخيصية في حالة حدوث خطأ

## استكشاف الأخطاء وإصلاحها

إذا واجهت مشاكل:

1. تأكد من تشغيل سكريبت التهيئة المناسب أولاً
2. تحقق من رسائل الخطأ في نافذة Console في Android Studio
3. قد تكون هناك مشكلة في الوصول إلى GitHub، تأكد من تكوين الشبكة بشكل صحيح

## طرق تنزيل بديلة

في حالة استمرار المشكلات، يمكنك تنزيل النماذج يدوياً:

1. قم بفتح ملف `assets-config.yml`
2. انسخ الروابط المذكورة فيه
3. قم بتنزيل كل ملف وضعه في المسار المناسب

## للمستخدمين المتقدمين

يمكنك تشغيل سكريبت التنزيل يدوياً:

```bash
python app/script/setup_and_download.py
```

هذا سيقوم بتثبيت المكتبات المطلوبة وتنزيل النماذج في خطوة واحدة.
